\# D{0-1}KP 背包问题求解系统

基于动态规划的 D{0-1}KP 优化求解工具，带GUI界面



\## 功能

1\. 读取 D{0-1}KP 标准数据

2\. 绘制重量-价值散点图

3\. 按第三项价值重量比非递增排序

4\. 动态规划求解最优解

5\. 导出结果 TXT  Excel



\## 运行环境

Python 3.x

matplotlib、pandas、tkinter



\## 运行方式

python srcmain.py
